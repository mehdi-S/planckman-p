# Workman-p Planck Layout

put these files in `qmk_firmware\keyboards\planck\keymaps\` folder and compile with the command 

> qmk compile -kb planck/rev6 -km planckman-p

in QMK MMSYS

more info : https://github.com/qmk/qmk_firmware

layout details:
